# Viso.Tracker Specifications

This folder is the technical source of truth for Viso.Tracker. All requirements here govern implementation. Documentation is in English.

- Lifecycle: Propose changes via PRs. Specs must remain concise, testable, and implementation-agnostic.
- Conventions: Use MudBlazor for all UI, CSS isolation (`.razor.css`), .NET 10 Blazor WASM, AOT + aggressive trimming, and WCAG 2.2 AA accessibility.
- Structure: Incremental, numbered specs (e.g., `0001-*.md`).
